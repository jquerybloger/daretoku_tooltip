jQuery.daretoku_tooltip

いまさら感があるtooltipのjQuerypluginです。
初めてjQuerypluginを作成しました。
なので色々と【もっとこうしたほうがいい】っていう所があると思います。
githubでコードを公開していますので、よろしければforkお願いします。
github

動作確認ブラウザ

IE7,8,9,Google chrome 19.0.1084.56 m,fire fox

導入に必要なもの

jQuery本体
jQuery easing

機能紹介

light boxとかにも対応
tooltipの横・縦幅を指定できる
表示する際のアニメーションを指定できる
表示位置を数値で指定できる
title,alt属性に対応

lightboxに対して使う場合

lightboxに対して使う場合は、ブログの方に記述してある通りにjQuery.lightbox.jsを編集してください。
指定も$("#hoge img").tooltip();のようにimgに指定してください。

リンク要素

リンクに対して指定した場合
リンク要素だけの場合はオプションのswichを"title_text"にしてください。
